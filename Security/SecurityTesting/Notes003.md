# CIA Triad - Basic High Level Objectives of Security Testing
- Basic or high level Goals and Objectives of Security Testing or Security Program is to achieve CIA Triad
- CIA Traid stands for - Confidentiality, Integrity and Availability
  Confidentiality - Information is private and securedIntegrity - Only Authorized people should be able to update (add,modify,delete) InformationAvailability - Application/System is available all the time to the intended users.
