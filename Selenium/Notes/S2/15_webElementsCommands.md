# Selenium WebDriver - WebElement Commands

## What is Web Element?
The term web element refers to a HTML element.
e.g. p tag, anchor tag

### Clear Command
```
clear():void
```
Command:  
`element.clear()`

### Sendkeys Command
Method:  
`sendKeys(CharSequence?KeysToSend):void;`

Command:
`element.sendKeys("text");`

### Click Command
Method:  
`click():void;`  

Command:  
`element.click()`
